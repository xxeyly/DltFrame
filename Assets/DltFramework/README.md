V1.1.5   Hierarchy新增
		 R!黄色:可以忽略,只是名称不一样,点击可自动更改名称
		 R!红色:不可忽略,继续使用会导致报错,名称不规范,需要手动更改名称
		 H!红色:不可忽略,继续使用会导致报错,子级中包含了热更脚本,点击H!会自动移除不规范内容
		 Hierarchy旧版
		 H:表示是热更
		 E:实体
		 U:表示要生成绑定脚本
		 S:场景组件
		 眼睛开:当前实体显示状态
		 眼睛闭:当前视图隐藏状态
		 开锁:当前视图全局操作时会忽略
		 解锁:当前视图全局操作时不会忽略
